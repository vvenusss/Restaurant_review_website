# Foodview
Foodview is a restaurant-centric review application that allows diners to browse restaurants, search listings, and manage their profiles. The platform also supports restaurant owners and administrators with role-based acess to different features.

This project focuses on both front-end and back-end functionality using PHP and SQL.

## Features

### User roles
1. Diners
    - Browse restaurants
    - Search listings
    - View and edit their profile
2. Restaurant Owner
    - Manage restaurant informatin
    - Add restaurants
    - View customer reviews
3. Admin
    - Manage users and restaurants
    - Moderate platform content
